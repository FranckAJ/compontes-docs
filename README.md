# Uso

Esta aplicação utiliza o Vuepress. Para instalação do mesmo é necessário a execuçação dos seguintes comandos:

```shell
yarn add -g vuepress 
 # OU 
npm install -g vuepress

```

Execução e build:

```shell
 # Execução local
vuepress dev

# Gerar Build para deploy
vuepress build

```
